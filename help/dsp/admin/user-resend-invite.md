# 为您的组织帐户重新发送待决用户邀请

*仅管理员用户*

您可以为组织帐户重新发送邀请，以向用户发送待定邀请。

1. 执行以下任一操作：

   * 在主菜单中，单击 **[!UICONTROL Settings]>[!UICONTROL Users]**.
   * 在主菜单中，单击 **[!UICONTROL Settings]>[!UICONTROL Account]**，然后单击 **[!UICONTROL Edit users]**.

1. 在 **邀请** ，单击 ![重新发送](/help/dsp/assets/resend.png).
